# Stack
Daleel uses the following stack:
- [BladeOne](https://github.com/EFTEC/BladeOne)
- [PHP Parser](https://github.com/nikic/PHP-Parser)
- [leage/commonmark](https://commonmark.thephpleague.com/)
- [commonmark-highlighter](https://github.com/spatie/commonmark-highlighter)
- [phpDocumentor](https://github.com/phpDocumentor/phpDocumentor)
- [PestPHP](https://pestphp.com/)
- [Symfony Components](https://symfony.com/components)
- [Bootstrap Icons](https://icons.getbootstrap.com/)
- [TailwindCSS](https://tailwindcss.com/)
